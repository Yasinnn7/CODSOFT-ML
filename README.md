## Projects Included

### 1️⃣ Movie Genre Classification

Created a text classification model to predict movie genres based on plot summaries.

#### 🔹 Objective
Automatically classify movies into genres using Natural Language Processing (NLP).

#### 🔹 Techniques Used
- Text Preprocessing
- TF-IDF Vectorization
- Naive Bayes
- Logistic Regression

#### 🔹 Evaluation Metrics
- Accuracy Score
- Precision
- Recall
- F1-Score

#### 🔹 Libraries Used
`Pandas` `NumPy` `NLTK` `Scikit-learn`
